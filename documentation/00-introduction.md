# Introduction to AWS Cognito:

AWS provides a managed solution, Amazon Cognito, for implementing authentication mechanisms in web applications used at St. Thomas. Using a managed service reduces the code base and, therefore, the maintenance and security risks compared to "rolling your own" authentication infrastructure.
